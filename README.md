# Simple Counter App

A lightweight web-based counter application built with HTML, CSS, and JavaScript.

## Features
- **Increase**: Increment the count value.
- **Decrease**: Decrement the count value.
- **Reset**: Reset the count back to zero.

## How to Run
Simply open the `index.html` file in any modern web browser.
